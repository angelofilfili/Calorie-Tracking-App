# Calorie-Tracking-App
Calorie Tracking App
My Chosen domain is habit tracker.
My goal is to create an app where you can add the nutritional facts of food you eat like calories, carbs, fats, protein, and more. I also want to allow the user to change their calorie goal and macro goals. I want the user to be able to manually add their food and its information so they are able to successfully track their calories for their goals.
